Add, Edit and Delete Resolver Component

If Add, Edit and Delete need to be resolved after confirmation from backend then resolve_add, resolve_edit and resolve_delete need to be set to true under actions config. And then listen to the rowadd, rowsave and rowdelete events. See below for more details.

Action

Below Configuration needs to be enabled. 
    {{"actions: {"}}
    {{"    resolve_add: true,"}}
    {{"    resolve_delete: true,"}}
    {{"    resolve_edit: true"}}
    {{"}"}}

Resolve

This is how action needs to be resolved. Then only modified record will be reflected in the grid.

    {{"onRowSave($e) {"}}
    {{"    const data = $e.data;"}}
    {{"    setTimeout(() => {"}}
    {{"        $e.resolve();"}}
    {{"    }, 1000);"}}
    {{"}"}}
